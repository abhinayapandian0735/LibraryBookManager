Library Book Manager (Java)

This is a simple Java console-based project that helps manage books in a library.
The program allows you to add books, issue books, return books, and view all available books.

Features

Add new books

Issue (borrow) books

Return books

Display all available books

How to Run

Save the file as LibraryManager.java

Open Terminal or Command Prompt

Navigate to the folder where the file is saved

Run the following commands:

javac LibraryManager.java
java LibraryManager
